# quiebre-administracion-directa
Artículo de investigación que evalúa si la directiva de la CGR sobre ejecución de obras por administración directa (vigente desde junio 2024) generó un cambio estructural en el uso de esa modalidad en la inversión pública peruana, usando datos de INFOBRAS y del Banco de Inversiones (MEF) con tests de quiebre estructural y DiD.
